# README

## requirements

- Ruby 3.0.1
- NodeJS >= 16.12.0
- yarn package manager
- library SQLite [halaman](https://www.sqlite.org/download.html)

## Installation

```bash
RAILS_ENV=production bin/rails db:migrate
RAILS_ENV=production bin/rails assets:precompile
```

## Tested Operating System

- Arch Linux
